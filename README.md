# German AI Chrome Extension

A Chrome extension that analyzes German news articles and extracts useful B2 vocabulary using OpenAI.

## Features

- Extracts 10 B2-level German vocabulary words
- Displays noun articles and plural forms
- Shows verb conjugations (Präsens, Präteritum, Perfekt)
- Provides Chinese explanations
- Includes example sentences with translations

## Architecture

Chrome Extension → Local Node.js Server → OpenAI API

## Tech Stack

- Chrome Extension (Manifest v3)
- Node.js + Express
- OpenAI API
